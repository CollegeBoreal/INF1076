## L'objectif :
se connecter a distance en SSH pour configurer un routeur 


## Les commandes a retenir:

a partir du pc distant :

taper la commande : ssh -l "username + ip address du routeur"

dans notre cas:

```
> ssh -l samir 192.168.1.254
password: samirssh
>enable 
password: samir
```
